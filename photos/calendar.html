How to Update This Calendar

Events are stored in calendar-data.js. Open that file in any text editor and add a line to the CALENDAR_EVENTS array: 

{ date: "2026-02-14",
  title: "Valentine's Party",
  type: "special" },

Save the file and reload the page — the calendar updates automatically.
